.. py:function:: cluster_gene_embeddings(embeds: numpy.ndarray, n_clusters: int = 2, adata: Optional[anndata.AnnData] = None, algo: str = 'kmeans', return_labels: bool = False, seed: int = 808) -> Optional[numpy.ndarray]

   Cluster gene embeddings using k-means or agglomerative clustering.

   :param embeds: Gene embedding matrix.
   :type embeds: ndarray
   :param n_clusters: Number of clusters. Default is 2.
   :type n_clusters: int, optional
   :param adata: Annotated data object containing observation metadata. Default is None.
   :type adata: AnnData, optional
   :param algo: Clustering algorithm. Default is "kmeans".
   :type algo: {'kmeans', 'agglomerative'}, optional
   :param return_labels: If True, return cluster labels. If False, return None. Default is False.
   :type return_labels: bool, optional
   :param seed: Random seed for reproducibility. Default is 808.
   :type seed: int, optional

   :returns: Cluster labels if return_labels is True, otherwise None.
   :rtype: ndarray or None

.. py:function:: subcluster(corr_array: numpy.ndarray, genes: Sequence[Any], distance_threshold: float = 0.2, n_subclusters: Optional[int] = None, plot_tree: bool = True, gene_names_ordered: Optional[Sequence[Any]] = None, metric: str = 'euclidean', linkage: str = 'ward') -> Dict

   Perform hierarchical clustering on gene embedding correlations.

   :param corr_array: Gene embedding correlation matrix.
   :type corr_array: ndarray
   :param genes: Gene names or identifiers corresponding to rows in corr_array.
   :type genes: array-like
   :param distance_threshold: Distance threshold for cluster merging in hierarchical clustering.
                              Default is 0.2.
   :type distance_threshold: float, optional
   :param n_subclusters: If specified, overrides distance_threshold and sets exact number of clusters.
                         Default is None.
   :type n_subclusters: int, optional
   :param plot_tree: If True, generate and display dendrogram visualization. Default is True.
   :type plot_tree: bool, optional
   :param gene_names_ordered: Ordered gene names for dendrogram visualization. Default is None.
   :type gene_names_ordered: array-like, optional
   :param metric: Distance metric for clustering. Default is "euclidean".
   :type metric: str, optional
   :param linkage: Linkage criterion for agglomerative clustering. Default is "ward".
   :type linkage: str, optional

   :returns: Dictionary mapping subcluster IDs to lists of genes in each subcluster.
   :rtype: dict
