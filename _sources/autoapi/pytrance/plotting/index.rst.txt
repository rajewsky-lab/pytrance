.. py:function:: embedding_pca(adata: anndata.AnnData, embeds_pca: numpy.ndarray, key: str = 'leiden', pca_model: Optional[sklearn.decomposition.PCA] = None, categorical: bool = True, **kwargs: Any) -> None

   Plot PCA embeddings with gene cluster coloring.

   :param adata: Annotated data object containing containing observation metadata.
   :type adata: AnnData
   :param embeds_pca: 2D PCA-reduced embeddings.
   :type embeds_pca: ndarray
   :param key: Column name in adata.var for cluster assignments. Default is "leiden".
   :type key: str, optional
   :param pca_model: Fitted PCA model. If provided, variance explained percentages are added
                     to axis labels. Default is None.
   :type pca_model: PCA, optional
   :param categorical: If True, color by categorical clusters using seaborn scatterplot.
                       If False, use continuous color mapping. Default is True.
   :type categorical: bool, optional
   :param \*\*kwargs: Additional keyword arguments passed to scatterplot/scatter functions.

   :returns: Displays plot using matplotlib.pyplot.show().
   :rtype: None

   Create multi-panel grid visualization of cells with spots/transcripts.

   :param n_rows: Number of rows in subplot grid.
   :type n_rows: int
   :param n_cols: Number of columns in subplot grid.
   :type n_cols: int
   :param adata: Annotated data object with transcript-level observations.
   :type adata: AnnData

   :type genes: array-like, optional
   :param cell_list: List of cell IDs to plot.

   :param cell_boundaries: Cell boundary coordinates (format depends on technology). Default is None.

   :param hue: Style to use for coloring transcripts ('gene', 'gene set', 'gene set only',
               or column name). Default is "gene".

   :param plot_type: Type of plot for each subplot. Default is "scatter".
   :type plot_type: {'scatter', 'histogram_absolute', 'histogram_relative'}, optional
   :param z: Z-slice value to filter transcripts. If None, uses all z values. Default is None.
   :type z: float, optional
   :param cell_key: Column name in adata.obs for cell IDs. Default is "cell".
   :type cell_key: str, optional
   :param gene_key: Column name in adata.obs for gene assignment. Default is "gene".
   :type gene_key: str, optional
   :param x_key: Column name in adata.obs for x coordinates. Default is "x".
   :type x_key: str, optional
   :param y_key: Column name in adata.obs for y coordinates. Default is "y".
   :type y_key: str, optional
   :param z_key: Column name in adata.obs for z coordinates. Default is "z".
   :type z_key: str, optional
   :param cmap: Colormap name for gene coloring (scatter plots). Default is "tab20".
   :type cmap: str, optional
   :param technology: Technology for boundary format interpretation. Default is None.
   :type technology: {'xenium', 'merfish'}, optional
   :param s: Marker size for scatter plots. Default is 50.
   :type s: float, optional
   :param scale: Scale factor for subplot dimensions. Default is 0.1.
   :type scale: float, optional
   :param col_padding: Horizontal padding between columns. Default is 10.
   :type col_padding: float, optional
   :param row_padding: Vertical padding between rows. Default is 10.
   :type row_padding: float, optional
   :param boundary_padding: Padding around data in axis limits. Default is 5.
   :type boundary_padding: float, optional
   :param fontsize: Font size for subplot titles. Default is 14.
   :type fontsize: int, optional
   :param return_fig: If True, return figure object. If False, display with plt.show(). Default is False.
   :type return_fig: bool, optional
   :param cbar: Show colorbar for histogram plots. Default is True.
   :type cbar: bool, optional
   :param \*\*kwargs: Additional keyword arguments passed to plotting functions.

.. py:function:: pairwise_clq_heatmap(gene_set: Sequence[Any], clq_scores: Dict, return_matrix: bool = False, return_axis: bool = False, **kwargs: Any) -> Union[pandas.DataFrame, matplotlib.axes.Axes, Tuple[pandas.DataFrame, matplotlib.axes.Axes], None]

   Visualize pairwise CLQ scores as heatmap.

   :param gene_set: List of gene names to include in the heatmap.
   :type gene_set: array-like
   :param clq_scores: Dictionary mapping gene pairs (tuples) to CLQ score values.
                      Expected keys are (gene1, gene2) tuples for all pairs in gene_set.
   :type clq_scores: dict
   :param return_matrix: If True, return the CLQ score matrix. Default is False.
   :type return_matrix: bool, optional
   :param return_axis: If True, return the heatmap axis/axes object. Default is False.
   :type return_axis: bool, optional
   :param \*\*kwargs: Additional keyword arguments passed to seaborn.heatmap().

   :returns: Depending on return_matrix and return_axis flags:
             - (False, False): None (displays plot)
             - (True, False): DataFrame with CLQ scores
             - (False, True): Heatmap axes object
             - (True, True): Tuple of (DataFrame, heatmap axes)
   :rtype: DataFrame or AxesSubplot or tuple or None
