.. py:function:: clq_pairwise(adata: anndata.AnnData, genes: Sequence[Any], graph: Optional[scipy.sparse.spmatrix] = None, radius: float = 2, n_neighbors: Optional[int] = None, n_permutations: int = 0, cell_key: str = 'cell', cat_key: str = 'gene', min_counts: int = 2, n_processes: int = 1, seed: int = 808, **kwargs: Any) -> Tuple[Dict, Dict]

   Compute co-localization quotient (CLQ) scores between gene pairs.

   :param adata: Annotated data object with transcript-level observations.
   :type adata: AnnData
   :param genes: List of gene names to compute pairwise CLQ for.
   :type genes: array-like
   :param graph: Precomputed spatial neighbor graph. If None, computed from coordinates.
                 Default is None.
   :type graph: sparse matrix, optional
   :param radius: Spatial radius for neighbor graph construction. Default is 2.
   :type radius: float, optional
   :param n_neighbors: Number of nearest neighbors if radius is None. Default is None.
   :type n_neighbors: int, optional
   :param n_permutations: Number of permutations for score adjustment. Default is 0.
   :type n_permutations: int, optional
   :param cell_key: Column name in adata.obs for cell IDs. Default is "cell".
   :type cell_key: str, optional
   :param cat_key: Column name in adata.obs for gene/category assignment. Default is "gene".
   :type cat_key: str, optional
   :param min_counts: Minimum transcript count per gene per cell to include in analysis.
                      Default is 2.
   :type min_counts: int, optional
   :param n_processes: Number of parallel processes for computation. Default is 1.
   :type n_processes: int, optional
   :param seed: Random seed for reproducibility. Default is 808.
   :type seed: int, optional
   :param \*\*kwargs: Additional keyword arguments passed to clq().

   :returns:

             - pairwise_clqs : dict
                 Dictionary mapping gene pairs to dictionaries containing 'clqs', 'clqs_adj',
                 and 'clqs_perm' (raw, adjusted, and permuted CLQ scores by cell).
             - aggregated_norm_clqs : dict
                 Aggregated statistics with 'mean' and 'median' keys, mapping gene pairs
                 to aggregated normalized CLQ scores.
   :rtype: tuple

   Compute co-localization quotient (CLQ) scores across all cells.

   :param adata: Annotated data object with transcript-level observations.
   :type adata: AnnData
   :param genes: Single gene name (str) for self co-localization or list of genes for
                 category definitions.
   :type genes: str or array-like
   :param graph: Precomputed spatial neighbor graph. If None, computed from coordinates.
                 Default is None.
   :type graph: sparse matrix, optional
   :param radius: Spatial radius for neighbor graph construction. Default is 2.
   :type radius: float, optional
   :param n_neighbors: Number of nearest neighbors if radius is None. Default is None.
   :type n_neighbors: int, optional
   :param n_permutations: Number of permutations for score adjustment. Default is 0.
   :type n_permutations: int, optional
   :param n_processes: Number of parallel processes. Default is 1.
   :type n_processes: int, optional
   :param cell_key: Column name in adata.obs for cell IDs. Default is "cell".
   :type cell_key: str, optional
   :param cat_key: Column name in adata.obs for gene/category assignment. Default is "gene".
   :type cat_key: str, optional
   :param x_key: Column name in adata.obs for x coordinates. Default is "x".
   :type x_key: str, optional
   :param y_key: Column name in adata.obs for y coordinates. Default is "y".
   :type y_key: str, optional
   :param z_key: Column name in adata.obs for z coordinates. Default is "z".
   :type z_key: str, optional
   :param verbose: Verbosity level (0=silent, 1+=with progress bars). Default is 1.
   :type verbose: int, optional
   :param seed: Random seed for permutation testing. Default is 808.
   :type seed: int, optional
   :param \*\*kwargs: Additional keyword arguments passed to clq_single_cell().

   :returns: If n_permutations > 0, returns tuple of:
             - cell_clqs : dict - Raw CLQ scores per cell
             - cell_clqs_adjusted : dict - Permutation-adjusted CLQ scores per cell
             - cell_clqs_permuted : dict - Permutation distributions per cell

             If n_permutations == 0, returns:
             - cell_clqs : dict - Raw CLQ scores per cell
   :rtype: dict or tuple

   Compute co-localization quotient (CLQ) for a single cell.

   :param cell_df: Cell-specific transcript data.
   :type cell_df: DataFrame
   :param cat_a: First gene category (list of gene names).
   :type cat_a: array-like
   :param cat_b: Second gene category (list of gene names).
   :type cat_b: array-like
   :param cat_key: Column name in cell_df for gene/category assignment.
   :type cat_key: str
   :param graph: Precomputed spatial neighbor graph. If None, constructed from coordinates.
                 Default is None.
   :type graph: sparse matrix, optional
   :param radius: Spatial radius for neighbor graph construction. Default is 2.
   :type radius: float, optional
   :param n_neighbors: Number of nearest neighbors if radius is None. Default is None.
   :type n_neighbors: int, optional
   :param n_permutations: Number of permutations for normalized CLQ computation. Default is None.
   :type n_permutations: int, optional
   :param x_key: Column name in cell_df for x coordinates. Default is "x".
   :type x_key: str, optional
   :param y_key: Column name in cell_df for y coordinates. Default is "y".
   :type y_key: str, optional
   :param z_key: Column name in cell_df for z coordinates. Default is "z".
   :type z_key: str, optional
   :param seed: Random seed for permutation testing. Default is 808.
   :type seed: int, optional
   :param \*\*kwargs: Additional keyword arguments passed to get_neighbors().

   :returns: If n_permutations is None or 0:
             - clq : float - Raw CLQ score

             - clq : float - Raw CLQ score

             - clq_perms : list - Distribution of CLQ scores from permutations
   :rtype: float or tuple


.. py:function:: clq_significance(cell_clqs: Dict, cell_clqs_permuted: Dict, percentile: float = 5) -> Tuple[List, Dict]

   Assess statistical significance of CLQ scores using permutation distributions.

   :param cell_clqs: Observed raw CLQ per cell, mapping cell IDs to float values.
   :type cell_clqs: dict
   :param cell_clqs_permuted: Permutation distributions per cell, mapping cell IDs to lists of
                              CLQs from permutations.
   :type cell_clqs_permuted: dict
   :param percentile: Percentile threshold for significance testing. A cell is significant if
                      its observed CLQ is beyond the [percentile, 100-percentile] range.
                      Default is 5.
   :type percentile: float, optional

   :returns:

             - significant_clq_cells : list - Cell IDs with significant CLQ scores
             - observed_vs_percentile : dict - Fold-change between observed CLQ and
               the nearest percentile threshold for each cell (1.0 for non-significant)
   :rtype: tuple
